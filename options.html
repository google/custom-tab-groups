Rules

Each line should be a rule definition, formatted like: regex=groupName. Navigating to a URL that matches regex will place the tab in the groupName tab group. If colors are defined below, the tab group will have the corresponding color.

If you use a valid color name as groupName, it will instead be placed in a colored group with no name.

Colors

Each line should be a group definition, formatted like: groupName=color, where color is one of grey, blue, red, yellow, green, pink, purple, or cyan.
